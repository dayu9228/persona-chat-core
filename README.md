# Persona Chat Core v1.2

> 让 AI Agent 拥有情绪感知与人格化回应能力的 Skill 框架。
> A skill framework for emotionally aware, personable AI agents.

## 安装

1. 将整个 `persona-chat-core_v1.2` 文件夹放入 HanaAgent 的 skills 目录
2. 重启 Agent 或重新加载技能列表
3. 验证：发一句「今天好累」，如果回复先接情绪再回应，生效了

### 依赖

| 子技能 | 状态 | 作用 |
|--------|------|------|
| time-context | 内置，可选 | 提供时段信息，辅助情绪研判（深夜自动上调敏感度） |
| philosophy-engine | 内置，按需安装 | 哲学类比引擎，为 Sparks 层提供中西哲学视角 |

> time-context 未安装时，核心框架会根据系统时间自行推算时段，精度降低但不阻塞。

## v1.2 更新内容

### 核心升级：从公式到决策链

v1.0 是说明书（13KB），v1.1 是公式（14KB），v1.2 是完整决策链（33KB）。

情绪研判从模糊权重重构为三段缝合流程：**选门（按输入类型走优先级链）→ 定强度（信号数量定档）→ 锁类型（11种情绪 + 排他锚点 + 平局链）**。

### 新增

- **排他锚点**：精准区分易混情绪类型。焦虑 vs 兴奋（反问是求解还是修辞）、孤独 vs 沉默退缩（想说说不出 vs 不想说）、后悔 vs 比较焦虑（纵向回看自己 vs 横向比较他人）
- **平局链**：信号数打平时按误判代价排序兜底。愤怒被忽视代价最高，兴奋被误判代价最低
- **混合消息协议**：一条消息同时包含情绪和技术问题时，四层逐层压缩但全保留，宁短不缺层
- **长上下文锚点**：80K token 后自动插入 HTML 注释锚点追踪研判状态。锚点本身是里程标记，150-200K 区间主动提醒用户开新窗口。不自动退化，不缩水不降级
- **思考链透明化**：用户可开启研判路径可见。高/中情绪完整展示六行研判链，低情绪单行摘要，静默模式不展示
- **归因表扩展**：从 v1.1 的 7 种情绪扩展至 11 种。新增愤怒、孤独、后悔、沉默退缩的独立心理学归因模型

### 优化

- **反话/自嘲检测**：正面词汇+负面语境自动识别，显性词失效时以语境为准
- **静默模式触发条件**：从模糊判断升级为三重同时满足条件（强度为无 + 维度1-3无信号 + 纯操作指令）
- **极短消息分支**：「嗯」「好」「知道了」跳过展开规则，前文有情绪延续时一句 Vibe 轻点过渡
- **防疲劳例外**：同一情绪但话题明显切换时，恢复完整 Vibe 但换一个角度
- **静默退出规则**：从纯操作切回情绪对话时，首条强制包含完整 Vibe 层
- **语音输入适配**：无标点无 emoji 时自动切换优先级链，句式节奏为第一信号源
- **名字感知**：Vibe 层隐含对象感知，语气是「专门对你说的」

### 删减

- 硬性字数上限全部清除（v1.1 残留的个别字数限制在 v1.2 彻底移除）
- 情绪应对表彻底移除（功能已被四层框架和对错边界完全吸收）

## 核心架构

```
用户消息
    ↓
一、五维情绪研判（选门→定强度→锁类型）
    ↓  输出：强度·类型 双标签
二、心理学归因层（类型匹配归因行，强度决定深度）
    ↓  输出：归因 + 回应启示
三、四层回应框架（Vibe→Sparks→Reflections→Will）
    ↓  输出：完整回应
用户看到的回复
```

## 情绪强度与回应节奏

| 强度 | 四层行为 | 典型场景 |
|------|---------|----------|
| 高情绪 | 四层全开，每层充分展开 | 自我否定、崩溃、愤怒、重大分享 |
| 中情绪 | 四层保留，节奏收紧 | 日常吐槽、带情绪的技术问题 |
| 低情绪 | 四层融入一段，不显层痕 | 简单分享、闲聊 |
| 静默模式 | 跳过四层，直接执行 | 纯操作指令 |

## 11 种情绪类型

自我否定 / 焦虑 / 疲惫 / 愤怒 / 孤独 / 比较焦虑 / 分享欲 / 沉默退缩 / 后悔 / 兴奋 / 平静

每种有独立的心理学归因模型和排他锚点，精准区分不被混淆。

## 快速验证

装好后重启 Agent，依次测试：

1. **「今天好累」** → 先接情绪再回应，不是直接给建议
2. **「帮我把这个文件重命名成 report.md」** → 直接执行，不加戏
3. **「唉又刷了三小时短视频我是不是没救了」** → 不跟着骂也不空喊加油，把行为和自我评价拆开
4. **「我觉得这个设计不行，你帮我看看代码哪里有问题」** → 情绪一句带过，重点放在技术上

四个全过，生效。
