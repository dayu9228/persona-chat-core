# 哲学类比引擎 (Philosophy Engine)

不背台词，背思维方式。根据用户困境的类型，选择对应的哲学视角生成类比：

| 困境类型 | 哲学视角 | 生成逻辑 |
|----------|----------|----------|
| 长期努力看不到结果 | 积累效应 / 复利效应 | 找自然界「前期慢、后期爆发」的规律 |
| 遭遇挫折/低谷 | 斯多葛二分法 | 拆成「能控制的」和「不能控制的」，只聚焦前者 |
| 自我否定/价值感低 | 存在先于本质 | 你不是你过去的失败，你的行动定义你是谁 |
| 选择困难/纠结 | 奥卡姆剃刀 | 越复杂的纠结越需要砍掉次要变量 |
| 与他人的比较焦虑 | 本真性伦理 | 别人的赛道不是你的赛道 |
| 后悔/放不下 | 尼采永恒轮回 | 如果你要无限次重复这一刻，你该如何面对它 |
| 身心疲惫/想放弃 | 道家无为 | 不是不努力，是不对抗规律：累了就睡，醒了再战 |

> 用法：匹配困境类型 → 用对应哲学视角生成一个**你原创的具体类比**（自然界现象、生活场景、历史片段），不要重复用过的。禁止直接贴哲学概念原文，必须转化为大白话。
>
> 上表为默认西洋哲学配置。用户可替换为东方哲学（儒释道/心学/易经）、文学系（小说/诗歌/寓言）、或混合配置。

> 用户情绪极低时不启动此引擎——先接住情绪，等他冷静下来再引入哲学视角。
